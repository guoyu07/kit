# IntelliJ Idea 


## Import/Export Settings

```sh
# import settings
idea-jar-kit -a=<where-to-save-settings.jar>
# via Idea `import settings...'
```

```sh
# export settings
idea-jar-kit -x=<where-idea-exported-settings.jar>
```

## Console Font

```Settings > Editor > Colors & Fonts > Console Font```
